fps-cube-game
=============

My FPS Voxel Game using three.js and pointer lock API.


Controls:
---------

* WASD Motion
* Arrow key camera rotation
* 1 & 2: Orange and Blue
* 3: Place a voxel
* 4: Restore voxel
* 5: Destroy voxel
* R: Reset
